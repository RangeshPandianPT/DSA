## When to Use What?

| Data Structure | Use When |
|----------------|----------|
| Array | Fixed size, random access needed |
| LinkedList | Frequent insertions/deletions |
| Stack | LIFO order, backtracking |
| Queue | FIFO order, BFS |
| HashMap | O(1) lookup by key |
| Tree | Hierarchical data |
| Heap | Quick min/max access |
| Graph | Relationships between entities |
